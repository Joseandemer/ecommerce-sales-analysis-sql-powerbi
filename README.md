# 📊 Análisis de Ventas y Clientes en E-commerce

## 📌 Descripción
Este proyecto analiza el comportamiento de ventas y clientes en un entorno e-commerce utilizando un modelo de datos relacional.

Se integraron múltiples tablas (customers, orders, order_items, products) para construir un modelo analítico que permita identificar patrones de consumo, productos más rentables y clientes clave.

---

## 🛠️ Tecnologías utilizadas
- Excel (limpieza y organización de datos)
- MySQL (consultas SQL y KPIs)
- Power BI (dashboard y visualización)
- DAX (medidas analíticas)
- Modelado estrella (relaciones entre tablas)

---

## 📊 KPIs principales
- Ganancia total generada
- Volumen total de productos vendidos
- Número total de clientes
- Categoría con mayor rentabilidad
- Top clientes por frecuencia de compra
- Ciudades con mayor volumen de ventas
- Productos con mayor rendimiento
- Tendencia de ganancias por mes

---

## 📈 Insights clave
- Electronics lidera la generación de ingresos.
- Existe dependencia en clientes de alto valor.
- Buenos Aires y Lima concentran mayor volumen de ventas.
- Se observa estacionalidad en los ingresos.
- Algunos productos destacan por su alta rentabilidad.

---

## 📖 Conclusión
El análisis permite identificar oportunidades de optimización en ventas, fidelización de clientes y expansión de mercado, contribuyendo a una mejor toma de decisiones basada en datos.
